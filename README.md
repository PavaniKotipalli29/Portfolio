# Pavani Kotipalli - Portfolio

A modern, responsive personal portfolio website built with React, Vite, and TypeScript. This project showcases my skills, projects, and professional experience with a focus on premium aesthetics and smooth user interactions.

## Technologies Used

- **Framework:** React 19
- **Build Tool:** Vite
- **Language:** TypeScript
- **Styling:** CSS3 (Custom Styles)
- **Animations:** Framer Motion

Follow these steps to set up the project locally on your machine.

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. **Clone the repository:**

   ```bash
   git clone <repository-url>
   cd pavani-kotipalli-portfolio
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Run the development server:**

   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:5173](http://localhost:5173) to view the site.
